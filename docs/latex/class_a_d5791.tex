Converts a three-\/byte message to voltage for the \mbox{\hyperlink{class_a_d5791}{AD5791}} DAC. 

This function converts a three-\/byte message received from the \mbox{\hyperlink{class_a_d5791}{AD5791}} DAC to a corresponding voltage value. It takes a {\ttfamily \mbox{\hyperlink{structspi__utils_1_1_message}{spi\+\_\+utils\+::\+Message}}} object as input, which contains the three bytes of the message. The function extracts the three bytes from the message and combines them to form a two\textquotesingle{}s complement decimal value using the {\ttfamily \mbox{\hyperlink{class_a_d5791_a5ec9971f907072befbe6c9f1be5aa933}{three\+Byte\+To\+Int()}}} function. It then calculates the voltage based on the decimal value using a specific formula for the \mbox{\hyperlink{class_a_d5791}{AD5791}} DAC. If the decimal value is less than or equal to 524287, the function calculates the voltage as the decimal value multiplied by the DAC full scale divided by 524287. Otherwise, if the decimal value is greater than 524287, the function calculates the voltage as the negation of (1048576 minus the decimal value) multiplied by the DAC full scale divided by 524288. The calculated voltage is returned by the function as a {\ttfamily double} value.

Converts an integer value to three separate bytes. 

This function converts the given integer value to three separate bytes. It takes the integer value as input and uses pointer parameters to store the resulting bytes. The function performs the following steps\+:
\begin{DoxyEnumerate}
\item Converts the integer value to three bytes by shifting and masking operations.
\begin{DoxyItemize}
\item The most significant byte is obtained by right-\/shifting the integer value by 16 bits and ORing it with 16 to set a control bit.
\item The middle byte is obtained by right-\/shifting the integer value by 8 bits and masking it with 255 (0x\+FF) to get the 8 least significant bits.
\item The least significant byte is obtained by masking the integer value with 255 (0x\+FF) to get the 8 least significant bits.
\end{DoxyItemize}
\item Stores the three resulting bytes in the memory locations pointed to by the DB1, DB2, and DB3 pointers.
\end{DoxyEnumerate}

Sets the voltage on the specified channel of the \mbox{\hyperlink{class_a_d5791}{AD5791}} DAC. 

This function sets the voltage on the specified channel of the \mbox{\hyperlink{class_a_d5791}{AD5791}} DAC. It takes the channel number, desired voltage, and an optional flag to update the analog outputs as input. The function performs the following steps\+:
\begin{DoxyEnumerate}
\item Begins a SPI transaction with the DAC settings.
\item Generates a SPI message using the {\ttfamily set\+Voltage\+Msg} function to set the desired voltage.
\item Sets the block size and number of blocks in the message to 3 and 1, respectively.
\item Checks if the desired voltage is within the valid range. If it exceeds the range, it prints an error message and returns 999.
\item If the voltage is within the valid range\+:
\begin{DoxyItemize}
\item For each block in the message, it brings the corresponding DAC sync pin LOW.
\item Transfers the message bytes using SPI.\+transfer to send the voltage data.
\item Sets the corresponding DAC sync pin HIGH to complete the transfer.
\end{DoxyItemize}
\item If the {\ttfamily update\+Outputs} flag is true, it calls the {\ttfamily update\+Analog\+Outputs} function to update the analog outputs.
\item Calculates the updated voltage based on the transferred message using the {\ttfamily bytes\+To\+Voltage} function and stores it in the {\ttfamily v\+Readings} array.
\item Returns the updated voltage.
\end{DoxyEnumerate}

\begin{DoxyEnumerate}

\item Shifts the extracted bits 8 positions to the left.
\item Combines the shifted bits with the second byte ({\ttfamily DB2}) using a bitwise OR operation.
\item Shifts the combined value 8 positions to the left.
\item Combines the shifted value with the third byte ({\ttfamily DB3}) using a bitwise OR operation.
\item Returns the resulting 32-\/bit integer value.
