This function performs a buffer ramp full reading from the \mbox{\hyperlink{class_a_d4115}{AD4115}} ADC by executing the necessary steps. First, it sets the ADC mode using the {\ttfamily \mbox{\hyperlink{class_a_d4115_aac4e474edfd743b06ade3ca7844b45a6}{adc\+Mode()}}} function. Then, it iterates through all 16 channels and performs the following steps for each active channel\+:
\begin{DoxyEnumerate}
\item Waits for the DRDY signal using the {\ttfamily \mbox{\hyperlink{class_a_d4115_aa8db977a1819b37f361fce54ecdfdc22}{wait\+Drdy()}}} function.
\item Reads the data from the ADC using the {\ttfamily \mbox{\hyperlink{class_a_d4115_a6fda2af1ec68c29a51feb23ba44eb0bc}{data\+Reading()}}} function.
\item Converts the read data to a decimal value using the {\ttfamily \mbox{\hyperlink{class_a_d4115_abdd6a1d6e51ef730162ab42f711ade70}{three\+Byte\+To\+Int()}}} function and stores it in the {\ttfamily \+\_\+channel\+Decimals} array.
\item Maps the decimal value to voltage using the {\ttfamily \mbox{\hyperlink{class_a_d4115_a3a38ddcf07b2523a40e6e45bf2065101}{voltage\+Map()}}} function and stores it in the {\ttfamily \+\_\+channel\+Voltages} array. After processing all active channels, the function sets the {\ttfamily \+\_\+adc\+Sync} pin to HIGH. Finally, it iterates through all active channels again and prints the channel number and corresponding voltage to the serial monitor using the {\ttfamily Serial.\+write()} function. The function returns 0 indicating successful execution.

Performs a test to read and print the configuration of the ADC channels. 

This function performs a test to read and print the configuration of the ADC channels. It reads the channel registers by sending specific message bits to the communications register of the ADC. The function iterates through all 16 channels and performs the following steps for each channel\+:
\begin{DoxyEnumerate}
\item Constructs the message bits by combining the communication register bits and the channel index.
\item Sends the message bits to the ADC using the {\ttfamily SPI.\+transfer()} function.
\item Reads the data bytes of the channel from the ADC and combines them to obtain a double precision value.
\item Prints the channel index and the read data bytes, as well as the combined value, to the serial monitor. After processing all channels, the function sets the {\ttfamily \+\_\+adc\+Sync} pin to HIGH and ends the SPI transaction. The function does not return a value. Returns the bits that configured each channel. Check data sheet to read this data.
